# Trail Running Platform — MVP

Shoe & Gear Coach: recomendador de calzado y equipo para corredores de trail.

## Segmento
Diego — corredor intermedio, planea primer ultra 50K, presupuesto limitado.

## MVP Scope
- Cuestionario de perfil (nivel, terreno, presupuesto, talla)
- Recomendacion de calzado/equipo
- Pagina de resultados con lista priorizada
